Data Cleanup

The script run_analysis.R cleans up the data via the following steps:

  1. Loads the training (train/X_train.txt) and test (test/X_test.txt) measurements

  2. Extracts only the measurements on the mean and standard deviation for each measurement using the feature names in features.txt

  3. Loads the subject data from train/subject_train.txt and test/subject_test.txt, and combines it with the measurements

  4. Loads the activity identifier from train/y_train.txt and test/y_test.txt and combines it with the measurements

  5. Loads the activity labels from activity_labels.txt and combines it with the measurements to ensure that the activity has a friendly label

  6. From the data set in step 5, creates a second, independent tidy data set with the average of each variable for each activity and each subject.

Code Book

The code book describing the tidy data produced is in CodeBook.md. Portions of the code book are generated using R code in the generate_codebook.R script.
